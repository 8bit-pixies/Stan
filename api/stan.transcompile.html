transcompile Module¶

The stan.transcompile module is the key module to convert a full script and run it

stan.transcompile.transcompile.transcompile(cstr)[source]¶

transcompile converts a stan script to python equivalent

cstr : the stan script in string format

Comments are stripped out first, and then all python code is ignored.

Multi-line comments start with /* and end with */ Single line comments start with * and end with ; or start with // Python code begin with #
